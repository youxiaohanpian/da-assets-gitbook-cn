# Asset UI

## 操作步骤

1. **Link to your project in Figma.** → 在 Figma 中链接到您的项目

2. **Open the list of cached projects that you have previously imported.**
   → 打开您之前导入的缓存项目列表

3. **Download the project from the link.**
   → 从链接下载项目，所有下载的项目都会自动缓存

4. **Import selected frames from the downloaded project.**
   → 从下载的项目中导入选定的帧

5. **Stop import.**
   → 停止导入

6. **Open asset settings.**
   → 打开资产设置

7. **Switch the asset display mode.** 可用的模式：
   - **In the inspector** — 在检查器中
   - **Windowed** — 窗口化

8. **Label displaying the current and latest version of the asset.**
   → 显示资产当前版本和最新版本的标签
   - **Stable** — 稳定版本
   - **Beta** — 尚不确定是否稳定或包含错误的版本
   - **Buggy** — 包含严重干扰资产运行的错误版本
