# Infinity Rotation（无限旋转）

## 设置步骤

### 步骤 1：添加组件

要为对象设置无限旋转，请为其添加 **RotationAnimator**，将 "**Looped**" 参数设置为 **360**（右旋转）或 **-360**（左旋转）。

### 步骤 2：配置动画

在 Animations 中启用 "**Looped**"，设置所需的曲线（例如，Linear 用于匀速旋转），并指定单次完整旋转的 **Duration**（持续时间）。

### 步骤 3：交互行为

如果在循环动画播放期间触发了另一个事件，循环动画将**暂停**，事件动画（例如，点击）将播放，然后循环动画将**恢复**。

## 参数说明

| 参数 | 说明 |
|------|------|
| **Looped** | 360 = 右旋转；-360 = 左旋转 |
| **Curve** | 动画曲线（如 Linear 匀速旋转） |
| **Duration** | 单次完整旋转的持续时间 |

## 注意事项

- 无限旋转适用于需要持续旋转的 UI 元素
- 支持与其他动画事件的交互
- 循环动画会在其他事件触发时智能暂停和恢复
