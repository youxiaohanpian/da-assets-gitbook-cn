# FCU UITK 介绍

## 简介

我强烈建议在使用此资源之前阅读本指南。

此资源是主资源 "Figma Converter for Unity" 的扩展。

一旦你在 "Figma Converter for Unity" 资源之上安装此扩展，它将获得将布局转换为 UI Toolkit 的能力。

此扩展不是 "Figma Converter for Unity" 的替代品，如果 "Figma Converter for Unity" 未安装，它将无法工作。

扩展与 FCU 的兼容性在最新版本级别上得到保证：扩展的最新版本保证可与 FCU 的最新版本配合使用。

宣传材料使用 "[Neomorphism Guide 2.0 | Original](https://www.figma.com/file/J1uPSOY5k577mDpSfGFven/Neomorphism-Guide-2.0-Original)" 模板，作者 Oleksandr Plyuto。
