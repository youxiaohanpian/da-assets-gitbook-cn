# D.A. Button（设计师文档）

## 功能描述

D.A. Button 是 D.A. Assets 提供的一个 Unity 组件，作为标准 Unity Button 的增强替代方案。

## 主要特性

| 特性 | 说明 |
|------|------|
| **多 TargetGraphic 支持** | 可以同时控制多个图形元素的颜色/精灵状态 |
| **AnimationCuve 动画** | 支持通过动画曲线实现颜色/大小/位置动画 |
| **精灵切换** | 支持不同状态下的精灵图切换 |
| **循环动画** | 支持循环动画效果 |

## 状态标签

与标准 Unity Button 类似，D.A. Button 也支持 Figma 中的状态标签：

| 标签 | 说明 |
|------|------|
| `def` | 默认状态 |
| `pres` | 按下状态 |
| `hov` | 悬停状态 |
| `dis` | 禁用状态 |

## 与 Unity Button 的区别

- Unity Button：只支持单个 TargetGraphic
- D.A. Button：支持多个 TargetGraphics，动画更灵活

## 获取方式

D.A. Button 是单独的资产包，需单独购买。
