# Json.NET - 通过 DLL 文件安装

如果通过 Package Manager 安装 Json.NET 不成功，请从官方仓库下载最新版本：

> https://github.com/JamesNK/Newtonsoft.Json/releases

<Aside type="caution">
不要使用 2020 年之前发布的 Json.NET 版本。
</Aside>

---

## 安装步骤

### 步骤 1
从官方仓库下载 Json.NET 最新版本：
https://github.com/JamesNK/Newtonsoft.Json/releases

> ⚠️ **重要提示：** 不要使用 2020 年之前发布的 Json.NET 版本。

### 步骤 2
**解压压缩包**，打开 **Bin** 文件夹，然后是 **netstandard2.0**，将 **Newtonsoft.Json.dll** 拖到项目中 **Assets** 文件夹。

### 步骤 3
安装 **Json.NET** 后，您可以使用 Package Manager 导入 Figma Converter for Unity。
