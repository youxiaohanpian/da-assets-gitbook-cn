# MCP Server - 用于 Figma 导入

## 概述

本指南介绍如何安装和配置 MCP 服务器以从 Figma 导入设计项目。

---

## 使用步骤

<Steps>

1. ### 步骤 1：安装 Python

   Python 是服务器运行的必要条件。

   不同操作系统的安装说明：
   - [macOS](https://www.youtube.com/watch?v=VJjeMdl6yWA)
   - [Windows](https://www.youtube.com/watch?v=C3bOxcILGu4)
   - [Linux](https://www.youtube.com/watch?v=IAco2SSuGms)

   <Aside type="note">
   MCP 服务器功能目前仅在 Windows 上进行过测试。\
   如果您是 macOS 或 Linux 用户，需要设置 MCP 服务器，请联系开发人员。
   </Aside>

2. ### 步骤 2：启用 IDE 支持

   启用您计划与 Converter MCP 服务器配合使用的 IDE 支持。

   在激活过程中，IDE 配置文件将被修改。配置文件将自动创建备份。

3. ### 步骤 3：启动服务器

   点击窗口底部的 **Start** 按钮启动服务器。

4. ### 步骤 4：开始导入

   在 IDE 中打开 Agent 对话框，并请求导入您的 Figma 项目。

   之后，Agent 将提示您选择特定页面或帧进行导入。

   按照 Agent 的进一步指示完成导入。\
   如果您在使用 Agent 时遇到任何困难，请通知开发人员。

</Steps>

---

## 系统要求

| 要求 | 说明 |
|------|------|
| **Python** | 必须安装 |
| **支持的 IDE** | Antigravity, Cursor, Codex, Coplay |
| **操作系统** | Windows（已测试）、macOS/Linux（需联系开发者） |

---

## 故障排除

| 问题 | 解决方案 |
|------|---------|
| 服务器无法启动 | 检查 Python 是否正确安装 |
| IDE 无法连接 | 确认 IDE MCP 支持已启用 |
| 导入失败 | 查看 IDE 控制台日志并联系开发人员 |
