# 脚本生成器 - 通过属性序列化

## 概述

如果您有自己编写的脚本，可以使用属性标记的方式来序列化和绑定 Figma 对象。

---

## 使用步骤

<Steps>

1. 如果您有自己的脚本，可以跳过生成步骤。

2. 将属性 `[FObject("frame_name")]` 添加到您的类，其中 **frame_name** 是 Figma 中此脚本所属的 frame 名称。

3. 将属性 `[FObject("component_name")]` 添加到您的序列化字段，其中 **component_name** 是 Figma 中组件的名称，其游戏对象应序列化到此字段。

4. 点击 **"Serialize Objects"** 按钮。

5. 游戏对象将根据您的设置序列化到字段中。

</Steps>

---

## 属性说明

| 属性 | 用途 | 示例 |
|------|------|------|
| `[FObject("frame_name")]` | 标记类所属的 Figma frame | `[FObject("HeaderFrame")]` |
| `[FObject("component_name")]` | 标记字段对应的 Figma 组件 | `[FObject("LogoImage")]` |

---

## 示例代码

```csharp
[FObject("MainPanel")]
public class MainPanelController : MonoBehaviour
{
    [FObject("TitleText")]
    private TextMeshProUGUI titleText;
    
    [FObject("BackgroundImage")]
    private Image backgroundImage;
    
    [FObject("SubmitButton")]
    private Button submitButton;
}
```
