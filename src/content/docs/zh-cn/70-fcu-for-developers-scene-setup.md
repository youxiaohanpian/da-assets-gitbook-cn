# 场景设置

## Step 1
前往 **File > Build Settings > Player Settings**。

（图片：Player Settings 路径）

## Step 2
找到 **Other Settings** 部分，将 **Active Input Handling** 设置为 **Both** 或 **Input System Package (New)**。

（图片：Active Input Handling 配置）

## Step 3
安装 **Input System** 包（如果尚未安装）。
