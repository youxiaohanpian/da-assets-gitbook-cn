# Scene Setup（场景设置）

## 步骤 1：导入资产

通过 **Unity Package Manager** 导入名为 **"Figma Converter for Unity"** 的资产。

## 步骤 2：创建空 GameObject 并添加脚本

在场景中创建一个**空的 GameObject**，然后在其上添加名为 **"FigmaConverterUnity"** 的脚本。

## 步骤 3：通过菜单创建资产（替代方式）

也可以通过 Unity 的**菜单**直接在场景中创建该资产（作为步骤 2 的替代方法）。
