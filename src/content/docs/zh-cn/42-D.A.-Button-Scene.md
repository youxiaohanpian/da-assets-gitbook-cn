# D.A. Button - 场景设置

## 场景设置 (Scene)

### 步骤 1：导入资产
通过 **Unity Package Manager** 导入该资产。

### 步骤 2：创建 GameObject 并添加脚本
在场景中创建一个**空的 GameObject**，并将 **"D.A. Button"** 脚本添加到它上面。

添加后，您将看到资源的 Inspector 界面。

![Asset Interface](/files/3uRW3wCwBHbU70Cy6e05)

*宽度: 292px*

### 步骤 3：序列化目标图形对象
将您想要制作动画的、带有 **Graphic 组件**的 GameObjects 序列化到 **TargetGraphics 数组**中（例如 **Image** 或 **Text**）。

所有位于带有 D.A. Button 脚本的父 GameObject 内部的 TargetGraphics 都将响应指针事件，例如：
- `IPointerClickHandler`
- `IPointerEnterHandler`
- `IPointerExitHandler`

![Demo Scene Screenshot](/files/LCW3K4g66Hcj5u3fehV9)

*截图展示了资源的演示场景，其中包含带有背景和按钮动画的弹窗。*
