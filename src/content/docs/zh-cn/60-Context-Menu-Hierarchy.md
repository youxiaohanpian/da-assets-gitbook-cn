# Context Menu - Hierarchy

## 概述

Hierarchy 上下文菜单提供了在 Unity 场景 Hierarchy 面板中直接操作 Figma Converter 对象的快捷功能。

---

## 可用操作

{% hint style="info" %}
Hierarchy 上下文菜单的完整功能列表和详细说明，请参考 **"Tools menu"** 文档。
{% endhint %}

---

## 常见操作

| 操作 | 说明 |
|------|------|
| 创建组件 | 在场景中创建 Figma Converter 组件 |
| 刷新 | 刷新选中的 Figma 对象 |
| 删除 | 从场景中移除选中的对象 |

---

## 使用方法

1. 在 Unity Hierarchy 面板中右键点击 Figma Converter 对象
2. 从上下文菜单中选择所需的操作
3. 等待操作完成
