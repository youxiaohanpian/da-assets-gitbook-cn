# Scene Backups and Project Cache（场景备份与项目缓存）

## 概述

本页面介绍 Figma Converter for Unity 的自动备份功能和项目缓存机制。

## 场景备份

### 自动备份

在进行某些操作（如创建预制体、更新布局）之前，资产会**自动创建场景备份**。

### 备份位置

备份文件存储在项目的 `Assets` 文件夹内的特殊目录中。

### 恢复备份

如需恢复某个备份：
1. 找到对应的备份文件
2. 将其拖入当前场景
3. 或手动复制备份内容到当前场景

## 项目缓存

### 缓存机制

- 从 Figma 下载的项目会被自动缓存
- 缓存允许您在不重新下载的情况下重新导入项目
- 缓存位置可在资产设置中配置

### 清除缓存

如果需要清除缓存：
1. 打开资产设置
2. 找到缓存管理选项
3. 选择清除缓存

> 💡 **提示**：清除缓存后，下次导入需要重新从 Figma 下载项目。

## 最佳实践

- 定期清理旧备份以节省磁盘空间
- 在重大更改前手动创建场景备份
- 确保缓存目录位于 Unity 的 Assets 文件夹内
