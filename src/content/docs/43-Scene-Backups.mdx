# 场景备份与项目缓存

<Steps>

1. 包含活动场景备份的文件夹位于：**Library\Backup\Scene**

2. 备份在每次导入前和创建预制件前自动创建。

3. 备份是针对**之前保存的本地场景文件**创建的。如果您在 Unity 界面中项目名称旁边看到星号 (*)，表示对场景所做的未保存更改不会包含在备份中。

4. 如果您从未保存过当前场景（场景文件不在磁盘上），场景将在导入前自动保存到路径 **"Assets/Scenes/time_scene_name.unity"**。

5. 每次下载项目时，Figma 项目中对象的变换和属性会被缓存。\
   要避免再次下载，您可以从下拉菜单中选择缓存版本。

   ![缓存选择界面](/files/wDTwqwwpmj2CwTvBbx34.png)

</Steps>

---

## 关键概念

| 概念 | 说明 |
|------|------|
| **备份位置** | `Library\Backup\Scene` |
| **自动备份触发** | 每次导入前、创建预制件前 |
| **备份条件** | 仅针对已保存的本地场景 |
| **未保存场景处理** | 自动保存到 `Assets/Scenes/time_scene_name.unity` |
| **项目缓存** | 避免重复下载 Figma 对象的变换和属性 |

---

## 注意事项

⚠️ 如果 Unity 界面中项目名称旁边有星号 (*)，表示未保存的场景更改不会包含在备份中。请记得在重要操作前保存场景。
