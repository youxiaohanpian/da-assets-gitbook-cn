# Json.NET

Json.NET（又名 Newtonsoft.Json）是一个必需的库，资产需要它才能正常运行；没有它，资产将无法工作，因为它负责解析来自 Figma 的 JSON 响应。

## 安装方法

有两种安装方法可供选择：

### 方法一：通过 Package Manager 安装

（详见子页面）

### 方法二：通过 DLL 文件安装

如果第一种方法不起作用，可以通过添加 DLL 文件到项目来实现。
