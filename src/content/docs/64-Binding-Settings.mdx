# Asset UI - Binding Settings

## 概述

Binding Settings（绑定设置）用于配置文本本地化组件与数据源之间的动态绑定关系。

---

## 绑定配置界面

<Steps>

1. 在本地化脚本中，将 **Localizator Key** 更改为 `label_current_data` 并设置 **Binding**。

   这意味着本地化脚本将**自动从 MonoBehaviour 脚本中的特定字段检索值**，并将其插入到本地化文本中，实时更新。

2. 启用 **"Binding Enabled"** 复选框以显示绑定设置。\
   然后，序列化包含要检索值的脚本的 GameObject。点击组合框并选择所需的值。\
   此过程类似于为 UI.Button 组件选择 OnClick 方法。

3. **"Argument Index"** 字段包含 [string.Format()](https://learn.microsoft.com/en-us/dotnet/api/system.string.format?view=netstandard-2.1) 的索引。\
   对于本地化 "Current value: {0}"，我们将值设置为 0，因为它对应于零索引的占位符。

   您可以将不同 GameObject 和脚本中的值绑定到单个本地化中，您也可以将同一字段绑定到不同的占位符，但**您不能将两个不同的字段绑定到同一占位符。**

   您可以绑定以下类型的私有、公共、静态和实例属性和字段：\
   string、int、float、double、long、short、byte、decimal。

4. 在 Playmode 中运行项目时，您会看到脚本 **InfinityIncrementInt.cs** 中的计数器在增加，文本组件无需任何额外代码即可显示其当前值。

5. 如果您想优化绑定，可以调整值更新频率，该频率由 **"Binding Rate Ms"** 字段控制。

   **如果设置两个不同的数字**，将使用 **Random.Range** 作为延迟，**如果设置两个相同的数字**，则使用固定延迟。

   该值以毫秒为单位设置。

</Steps>

---

## 支持的数据类型

| 类型 | 示例 |
|------|------|
| string | "Hello World" |
| int | 42 |
| float | 3.14f |
| double | 3.14159265359 |
| long | 1234567890L |
| short | 32767 |
| byte | 255 |
| decimal | 99.99m |

---

## 绑定示例

**本地化文本：** `Current value: {0}`

**绑定配置：**
- Argument Index: 0
- 绑定到: `InfinityIncrementInt.cs` 的 `counter` 字段

**结果：** 文本将实时显示计数器的当前值
