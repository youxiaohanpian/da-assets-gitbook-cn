# D.A. Localizator

## 概述

**D.A. Localizator** 是 D.A. Assets 开发的专业本地化解决方案，专为 Unity 项目设计，可与 Figma Converter 无缝集成。

<Aside type="note">
D.A. Localizator 提供了一种轻量级、高效的本地化方式，适合不需要完整 I2 Localization 功能的项目。
</Aside>

---

## 核心特性

| 特性 | 说明 |
|------|------|
| **简单集成** | 与 Figma Converter 一键集成 |
| **轻量级** | 占用资源少，性能优异 |
| **实时预览** | 编辑器内实时预览本地化效果 |
| **多平台支持** | 支持所有主流 Unity 平台 |
