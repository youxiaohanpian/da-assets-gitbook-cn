# Auth（认证）

## 认证方式

与 Figma 进行身份验证/授权有两种方式：

1. **通过浏览器授权 (Browser)** - 通过浏览器完成身份验证流程
2. **通过粘贴令牌授权 (Token)** - 通过粘贴令牌进行身份验证
