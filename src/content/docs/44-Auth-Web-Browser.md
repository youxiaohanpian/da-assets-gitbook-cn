# 认证 - 通过 Web 浏览器

## 概述

本节介绍如何使用 Web 浏览器方式完成 Figma 认证。

{% hint style="info" %}
此认证方式需要您的计算机能够访问 Figma 网站。
{% endhint %}

---

## 认证步骤

### 步骤 1：启动认证流程
在 Figma Converter 界面中，选择 **"Auth with web browser"** 选项。

### 步骤 2：打开授权页面
系统将自动打开您的默认 Web 浏览器，导航到 Figma 授权页面。

### 步骤 3：登录 Figma 账户
如果尚未登录，请使用您的 Figma 账户凭据登录。

### 步骤 4：授权访问
在 Figma 授权页面上，点击 **"Allow"** 或 **"Authorize"** 按钮，允许 Figma Converter 访问您的 Figma 项目。

### 步骤 5：完成认证
授权成功后，关闭浏览器窗口并返回 Figma Converter。认证状态将自动更新。

---

## 故障排除

| 问题 | 解决方案 |
|------|---------|
| 浏览器未打开 | 检查系统默认浏览器设置 |
| 授权页面加载失败 | 检查网络连接和代理设置 |
| 授权超时 | 重试认证流程 |
